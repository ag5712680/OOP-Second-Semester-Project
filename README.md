# Sale Management System 
 # Overview
This project was developed during my second semester to master Object-Oriented Programming (OOP) and software architecture. It focuses on creating a structured, efficient way to handle sales records and basic business operations.

# Technologies Used
# Language: Java

# Database: MySQL / SQL

# Architecture: Object-Oriented Programming (OOP)

# Database Connectivity: JDBC 

# IDE: Eclips

 # Key Features
SQL Persistence: Integrates a database to ensure all sales records are saved permanently.

Record Management: Full CRUD functionality to Add, Update, and Remove sales entries.

Reporting: Automated generation of sales reports for data tracking.

Error Handling: Robust debugging and validation for user inputs.

# Learning Outcomes
Applied core OOP principles like Encapsulation, Abstraction, and Inheritance.

Learned to connect Java applications with relational databases using SQL.

Improved skills in structured coding, problem-solving, and logic building.
